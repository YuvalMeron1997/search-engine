# Mini Search Engine

A simple search engine implementing an inverted index and TF-IDF ranking.

## Features
- Inverted index construction
- TF-IDF ranking
- Simple query interface via Flask API

## Data
Uses a small sample dataset for demonstration purposes.

## Run

```bash
pip install -r requirements.txt
python app.py
```

Example:

http://127.0.0.1:5000/search?query=machine+learning
